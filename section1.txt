1. Introduction :
-------------------------------------------------------------------------------------------
Python is a (General Purpose) (High Level) (Dynamically Typed) (Programming Language).

Guido Van Rossum is the Father of Python.
He developed it in 1989 and was made public on 20th Feb 1991.
He developed it while working in National Research Institute (NRI) in Netherlands.

It is a very simple language to write. Less code required.

The name Python was given from the show which Guido van Rossum liked very much.
The show was "The Complete Monty Python Flying Circus" which was broad casted on BBC
from 1969 to 1974.

5. Flavors of Python :
-------------------------------------------------------------------------------------------
It basically means all the special libraries that have been created to interact with other
programming languages and for particular jobs. For example :
1. CPython - For working with C Language related application.
2. JPython - For working with Java Language related application.
3. Ruby Python - For working with Ruby Language related application.
4. Anaconda Python - For working with ML related application.
etc etc etc.
